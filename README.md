# Smart Finance AI

Smart Finance AI is a premium, dark-first personal finance dashboard built with Next.js App Router, TypeScript, Tailwind CSS, and Framer Motion.

## What it does

- Track expenses in real time
- Organize spending by month
- Search and delete entries instantly
- Persist data locally in the browser
- Open the expense tracker in a popup first

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Framer Motion
- next-themes
- Lucide React

## Features

- Dark-first UI
- Month-based expense tracking
- Live category breakdowns
- Local storage persistence
- Responsive layout
- SEO metadata and structured data

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Build for Production

```bash
npm run build
```

## Deploy to Vercel

1. Push the repo to GitHub.
2. Import the repo into Vercel.
3. Keep the default Next.js settings.
4. Deploy.
